# FoodTracker
This is a sample project for IOS Mobile Application Development using XCode and Swift programming language. 

Highlights
* Basic UI Design implementation
* Connect UI with Actions
* Data Model
* Navigation
* Table View List
* Edit & Delete behaviours
* Data Persist
